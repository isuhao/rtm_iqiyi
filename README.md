## 如何使用

* 右键点击 "下方的按钮" ，选择右键菜单中的 "复制链接地址" ，然后手动添加书签：随便写个名字，地址粘贴复制的内容。
* Ctrl + B 显示书签栏后，直接将 "下方的按钮" 拖拽到浏览器书签栏。
